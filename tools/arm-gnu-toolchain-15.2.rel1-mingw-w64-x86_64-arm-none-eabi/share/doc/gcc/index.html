Introduction ¶

This manual documents how to use the GNU compilers, as well as their features and incompatibilities, and how to report bugs. It corresponds to the compilers (Arm GNU Toolchain 15.2.Rel1 (Build arm-15.86)) version 15.2.1. The internals of the GNU compilers, including how to port them to new targets and some information about how to write front ends for new languages, are documented in a separate manual. See Introduction in GNU Compiler Collection (GCC) Internals.
