close ¶

Synopsis:
int close(int fd);
Request:

‘Fclose,fd’

Return value:

close returns zero on success, or -1 if an error occurred.

Errors:
EBADF

fd isn’t a valid open file descriptor.

EINTR

The call was interrupted by the user.
