27.5.7 GDB/MI Ada Exception Information ¶

Whenever a *stopped record is emitted because the program stopped after hitting an exception catchpoint (see Setting Catchpoints), GDB provides the name of the exception that was raised via the exception-name field. Also, for exceptions that were raised with an exception message, GDB provides that message via the exception-message field.
