G.5.13 RX Features ¶

The ‘org.gnu.gdb.rx.core’ feature is required for RX targets. It should contain the registers ‘r0’ through ‘r15’, ‘usp’, ‘isp’, ‘psw’, ‘pc’, ‘intb’, ‘bpsw’, ‘bpc’, ‘fintv’, ‘fpsw’, and ‘acc’.
