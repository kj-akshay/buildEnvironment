8.169 LEN_TRIM — Length of a character entity without trailing blank characters ¶

Synopsis:

RESULT = LEN_TRIM(STRING [, KIND])

Description:

Returns the length of a character string, ignoring any trailing blanks.

Class:

Elemental function

Arguments:
STRINGShall be a scalar of type CHARACTER, with INTENT(IN)
KIND(Optional) A scalar INTEGER constant expression indicating the kind parameter of the result.
Return value:

The return value is of type INTEGER and of kind KIND. If KIND is absent, the return value is of default integer kind.

Standard:

Fortran 90 and later, with KIND argument Fortran 2003 and later

See also:

LEN — Length of a character entity,
ADJUSTL — Left adjust a string,
ADJUSTR — Right adjust a string
