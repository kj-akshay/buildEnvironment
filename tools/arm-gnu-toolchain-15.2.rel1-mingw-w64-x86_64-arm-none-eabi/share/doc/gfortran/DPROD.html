8.88 DPROD — Double product function ¶

Synopsis:

RESULT = DPROD(X, Y)

Description:

DPROD(X,Y) returns the product X*Y.

Class:

Elemental function

Arguments:
XThe type shall be REAL.
YThe type shall be REAL.
Return value:

The return value is of type REAL(8).

Example:
program test_dprod
    real :: x = 5.2
    real :: y = 2.3
    real(8) :: d
    d = dprod(x,y)
    print *, d
end program test_dprod
