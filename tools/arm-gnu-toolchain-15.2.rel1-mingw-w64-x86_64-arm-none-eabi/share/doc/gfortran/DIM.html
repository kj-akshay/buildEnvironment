8.86 DIM — Positive difference ¶

Synopsis:

RESULT = DIM(X, Y)

Description:

DIM(X,Y) returns the difference X-Y if the result is positive; otherwise returns zero.

Class:

Elemental function

Arguments:
XThe type shall be INTEGER or REAL
YThe type shall be the same type and kind as X. (As a GNU extension, arguments of different kinds are permitted.)
Return value:

The return value is of type INTEGER or REAL. (As a GNU extension, kind is the largest kind of the actual arguments.)

Example:
program test_dim
    integer :: i
    real(8) :: x
    i = dim(4, 15)
    x = dim(4.345_8, 2.111_8)
    print *, i
    print *, x
end program test_dim
