8.95 ERF — Error function ¶

Synopsis:

RESULT = ERF(X)

Description:

ERF(X) computes the error function of X.

Class:

Elemental function

Arguments:
XThe type shall be REAL.
Return value:

The return value is of type REAL, of the same kind as X and lies in the range -1 \leq erf (x) \leq 1 .

Example:
program test_erf
  real(8) :: x = 0.17_8
  x = erf(x)
end program test_erf
