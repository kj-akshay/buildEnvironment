8.59 CHAR — Character conversion function ¶

Synopsis:

RESULT = CHAR(I [, KIND])

Description:

CHAR(I [, KIND]) returns the character represented by the integer I.

Class:

Elemental function

Arguments:
IThe type shall be INTEGER.
KIND(Optional) A scalar INTEGER constant expression indicating the kind parameter of the result.
Return value:

The return value is of type CHARACTER(1)

Example:
program test_char
    integer :: i = 74
    character(1) :: c
    c = char(i)
    print *, i, c ! returns 'J'
end program test_char
Specific names:
