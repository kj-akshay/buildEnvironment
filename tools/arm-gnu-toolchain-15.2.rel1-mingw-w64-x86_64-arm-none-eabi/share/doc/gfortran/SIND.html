8.253 SIND — Sine function, degrees ¶

Synopsis:

RESULT = SIND(X)

Description:

SIND(X) computes the sine of X in degrees.

Class:

Elemental function

Arguments:
XThe type shall be REAL.
Return value:

The return value has same type and kind as X, and its value is in degrees.

Example:
program test_sind
  real :: x = 0.0
  x = sind(x)
end program test_sind
