8.128 GETLOG — Get login name ¶

Synopsis:

CALL GETLOG(C)

Description:

Gets the username under which the program is running.

Class:

Subroutine

Arguments:
CShall be of type CHARACTER and of default kind.
Return value:

Stores the current user name in C. (On systems where POSIX functions geteuid and getpwuid are not available, and the getlogin function is not implemented either, this returns a blank string.)

Example:
PROGRAM TEST_GETLOG
  CHARACTER(32) :: login
  CALL GETLOG(login)
  WRITE(*,*) login
END PROGRAM
Standard:

GNU extension
