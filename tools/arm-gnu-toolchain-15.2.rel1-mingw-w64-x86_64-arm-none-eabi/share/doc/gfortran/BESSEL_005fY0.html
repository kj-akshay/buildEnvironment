8.43 BESSEL_Y0 — Bessel function of the second kind of order 0 ¶

Synopsis:

RESULT = BESSEL_Y0(X)

Description:

BESSEL_Y0(X) computes the Bessel function of the second kind of order 0 of X. This function is available under the name BESY0 as a GNU extension.

Class:

Elemental function

Arguments:
XThe type shall be REAL.
Return value:

The return value is of type REAL. It has the same kind as X.

Example:
program test_besy0
  real(8) :: x = 0.0_8
  x = bessel_y0(x)
end program test_besy0
