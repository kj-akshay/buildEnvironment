8.254 SINH — Hyperbolic sine function ¶

Synopsis:

RESULT = SINH(X)

Description:

SINH(X) computes the hyperbolic sine of X.

Class:

Elemental function

Arguments:
XThe type shall be REAL or COMPLEX.
Return value:

The return value has same type and kind as X.

Example:
program test_sinh
  real(8) :: x = - 1.0_8
  x = sinh(x)
end program test_sinh
