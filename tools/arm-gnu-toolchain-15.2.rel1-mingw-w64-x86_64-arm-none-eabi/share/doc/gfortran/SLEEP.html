8.257 SLEEP — Sleep for the specified number of seconds ¶

Synopsis:

CALL SLEEP(SECONDS)

Description:

Calling this subroutine causes the process to pause for SECONDS seconds.

Class:

Subroutine

Arguments:
SECONDSThe type shall be of default INTEGER.
Example:
program test_sleep
  call sleep(5)
end
Standard:

GNU extension
