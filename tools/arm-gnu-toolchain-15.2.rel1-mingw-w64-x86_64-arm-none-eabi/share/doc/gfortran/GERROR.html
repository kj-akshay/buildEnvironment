8.120 GERROR — Get last system error message ¶

Synopsis:

CALL GERROR(RESULT)

Description:

Returns the system error message corresponding to the last system error. This resembles the functionality of strerror(3) in C.

Class:

Subroutine

Arguments:
RESULTShall be of type CHARACTER and of default kind.
Example:
PROGRAM test_gerror
  CHARACTER(len=100) :: msg
  CALL gerror(msg)
  WRITE(*,*) msg
END PROGRAM
Standard:

GNU extension

See also:

IERRNO — Get the last system error number,
PERROR — Print system error message
