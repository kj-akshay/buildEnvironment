8.72 CONJG — Complex conjugate function ¶

Synopsis:

Z = CONJG(Z)

Description:

CONJG(Z) returns the conjugate of Z. If Z is (x, y) then the result is (x, -y)

Class:

Elemental function

Arguments:
ZThe type shall be COMPLEX.
Return value:

The return value is of type COMPLEX.

Example:
program test_conjg
    complex :: z = (2.0, 3.0)
    complex(8) :: dz = (2.71_8, -3.14_8)
    z= conjg(z)
    print *, z
    dz = dconjg(dz)
    print *, dz
end program test_conjg
