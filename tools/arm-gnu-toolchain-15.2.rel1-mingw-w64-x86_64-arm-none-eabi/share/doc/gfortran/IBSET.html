8.142 IBSET — Set bit ¶

Synopsis:

RESULT = IBSET(I, POS)

Description:

IBSET returns the value of I with the bit at position POS set to one.

Class:

Elemental function

Arguments:
IThe type shall be INTEGER or UNSIGNED.
POSThe type shall be INTEGER.
Return value:

The return value is of the same type as I.

Specific names:
