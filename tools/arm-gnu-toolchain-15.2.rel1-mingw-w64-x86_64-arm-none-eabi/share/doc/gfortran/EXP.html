8.102 EXP — Exponential function ¶

Synopsis:

RESULT = EXP(X)

Description:

EXP(X) computes the base e exponential of X.

Class:

Elemental function

Arguments:
XThe type shall be REAL or COMPLEX.
Return value:

The return value has same type and kind as X.

Example:
program test_exp
  real :: x = 1.0
  x = exp(x)
end program test_exp
