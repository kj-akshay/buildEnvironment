8.51 BTEST — Bit test function ¶

Synopsis:

RESULT = BTEST(I, POS)

Description:

BTEST(I,POS) returns logical .TRUE. if the bit at POS in I is set. The counting of the bits starts at 0.

Class:

Elemental function

Arguments:
IThe type shall be INTEGER or UNSIGNED.
POSThe type shall be INTEGER.
Return value:

The return value is of type LOGICAL

Example:
program test_btest
    integer :: i = 32768 + 1024 + 64
    integer :: pos
    logical :: bool
    do pos=0,16
        bool = btest(i, pos) 
        print *, pos, bool
    end do
end program test_btest
