8.252 SIN — Sine function ¶

Synopsis:

RESULT = SIN(X)

Description:

SIN(X) computes the sine of X.

Class:

Elemental function

Arguments:
XThe type shall be REAL or COMPLEX.
Return value:

The return value has same type and kind as X.

Example:
program test_sin
  real :: x = 0.0
  x = sin(x)
end program test_sin
